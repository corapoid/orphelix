# Run Tests

Run all tests (unit + E2E) and report results with actionable feedback.

## Usage

```
/run-tests
```

or with filter:

```
/run-tests services
```

## What This Command Does

1. Runs unit tests (Vitest)
2. Runs E2E tests (Playwright)
3. Reports coverage
4. Identifies failures with suggested fixes
