# orphelix status

Display the current status of Orphelix application.

## Usage

```bash
orphelix status [options]
```

## Options

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--name` | `-n` | Instance name | `orphelix` |

## Example Output

```bash
$ orphelix status

┌────────────┬────┬──────┬────────┬─────┬─────────┐
│ name       │ id │ mode │ status │ cpu │ memory  │
├────────────┼────┼──────┼────────┼─────┼─────────┤
│ orphelix   │ 0  │ fork │ online │ 0%  │ 85.2 MB │
└────────────┴────┴──────┴────────┴─────┴─────────┘
```

## Status Values

- **online** - Running normally
- **stopped** - Stopped via `orphelix stop`
- **errored** - Crashed or error state
- **launching** - Starting up
