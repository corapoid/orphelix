# orphelix restart

Restart the Orphelix application without stopping pm2.

## Usage

```bash
orphelix restart [options]
```

## Options

| Option | Alias | Description |
|--------|-------|-------------|
| `--port` | `-p` | Custom port |
| `--name` | `-n` | Instance name |

## Example

```bash
orphelix restart
orphelix restart --port 8080
orphelix restart --name prod
```
