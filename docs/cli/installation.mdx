# Installation

## Prerequisites

Before installing Orphelix CLI, ensure you have:

- **Node.js** >= 24.0.0 ([Download](https://nodejs.org/))
- **kubectl** installed and configured ([Install Guide](https://kubernetes.io/docs/tasks/tools/))
- **npm** or **yarn** package manager
- **macOS** or **Linux** operating system

## Install via npm

Install Orphelix CLI globally using npm:

```bash
npm install -g orphelix
```

This will make the `orphelix` command available system-wide.

## Verify Installation

Check that the installation was successful:

```bash
orphelix --version
```

You should see output like:
```
Orphelix v1.0.0
```

## First Run

Start Orphelix for the first time:

```bash
orphelix start
```

## Configuration Options

### Custom Port

Run on a different port:

```bash
# Via command flag
orphelix start --port 8080

# Via environment variable
export ORPHELIX_PORT=8080
orphelix start
```

### Skip Hostname Configuration

If you don't want to configure `orphelix.local`, answer "No" when prompted:

```
Configure custom hostname orphelix.local? [Y/n]: n
â­ï¸  Skipped hostname configuration
```

You can still access via `http://localhost:3000`

## Troubleshooting Installation

### "Node version too old"

Upgrade to Node.js 24 or higher:

```bash
# Using nvm
nvm install 24
nvm use 24

# Or download from nodejs.org
```

### "kubectl not found"

Install kubectl:

```bash
# macOS
brew install kubectl

# Linux (Ubuntu/Debian)
sudo apt-get install kubectl

# Or follow official guide
# https://kubernetes.io/docs/tasks/tools/
```

### "Permission denied" during npm install

Run with sudo (not recommended) or fix npm permissions:

```bash
# Fix npm permissions (recommended)
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Then install without sudo
npm install -g orphelix
```

### "Port 3000 already in use"

Use a different port:

```bash
orphelix start --port 8080
```

Or kill the process using port 3000:

```bash
lsof -ti:3000 | xargs kill
```

### Database initialization failed

Remove the database and restart:

```bash
orphelix stop
rm orphelix.db*
orphelix start
```

## Uninstallation

To completely remove Orphelix:

```bash
# Stop and remove from pm2
orphelix delete

# Uninstall globally
npm uninstall -g orphelix

# Remove data (optional)
rm -rf ~/orphelix-backups
```
