## What is Orphelix?

Orphelix is a comprehensive web dashboard for monitoring and managing Kubernetes resources. Built with modern web technologies, it provides an intuitive interface for DevOps teams and developers to interact with their Kubernetes clusters.

## Why Orphelix?

<AccordionGroup>
  <Accordion icon="rocket" title="Easy to Use">
    Clean, modern interface designed for both beginners and experienced Kubernetes users. Start with demo mode to explore features without a cluster.
  </Accordion>
  <Accordion icon="shield" title="Secure">
    Uses your local kubeconfig for authentication. Secret values are never displayed. GitHub App integration provides granular repository permissions.
  </Accordion>
  <Accordion icon="bolt" title="Fast & Responsive">
    Built with Next.js 15, React 19, and Material-UI for optimal performance. Real-time updates without page refreshes.
  </Accordion>
  <Accordion icon="code" title="Developer Friendly">
    Full TypeScript support, comprehensive API documentation, and extensible architecture. Source-available under BUSL-1.1 (Apache-2.0 on 2030-01-01).
  </Accordion>
</AccordionGroup>
